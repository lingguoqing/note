# ubuntu 系统

### 配置静态IP地址

- 创建 **50-cloud-init.yaml** 文件 

  - 在 **/etc/netplan** 目录下创建

  ```cmd
  sudo touch 50-cloud-init.yaml
  ```

  - 修改文件内容 （sudo vim  50-cloud-init.yaml）

  ```yaml
  network:
      ethernets:
          enp4s1:
              addresses:
              - 10.51.200.13/24  # 对应这台设备的IP地址
              nameservers:
                  addresses:
                  - 223.5.5.5
                  search: []
              routes:
              -   to: default
                  via: 10.51.200.254 # 对应网关IP地址
      version: 2
  ```

  - 保存
    - 按 Esc 键后，英文输入 :wq! 进行保存
  - 应用 Netplan 配置

  ```cmd
  sudo netplan apply
  ```

  - 配置IP后查看是否更改成功

  ```cmd
  ip addr
  ```

### 如果出现磁盘未挂载完整，使用下面命令

```cmd
sudo vgdisplay ubuntu-vg # 确认卷组中还有剩余空间
sudo lvextend -l +100%FREE /dev/ubuntu-vg/ubuntu-lv # 扩展逻辑卷到剩余所有空间
sudo resize2fs /dev/ubuntu-vg/ubuntu-lv #  扩展 ext4 文件系统
df -h # 进行验证 ，在操作前可以先验证下是否全部挂载
```





## 解决平台上不显示IP地址问题

##### 在平台上点击挂载Tools

![image-20260513161049050](./%E6%90%AD%E5%BB%BA%E6%9C%8D%E5%8A%A1%E5%99%A8.assets/image-20260513161049050.png)

##### 操作命令

```cmd
mkdir xvdd
sudo mount /dev/sr0 xvdd/
cp vmtools-3.0.0.063.tar.bz2 /root/
sudo su # 切换用户
cd /root
apt update && apt install bzip2 -y  # 安装 bzip2
tar -jxvf vmtools-3.0.0.063.tar.b2  # 解压
cd vmtools
./install
```
